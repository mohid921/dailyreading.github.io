How you can get rid of Windows Bloatware with a single command




Windows has always had problems with bloatware (though Samsung recently claimed the crown in that area). Bloatware is a general term for applications, services, and programs that you do not need and never require, and Windows comes pre-installed with a large number of them.

Things got better with the launch of Windows 11 and Microsoft started cutting the fat off its installation media. But that's still a problem, with apps like Skype, 3DBuilder, Asphalt 8, Microsoft News, Feedback Hub, and more installed (and taking up space on your system) by default.

Removing bloatware helps you in two ways: you'll reclaim storage space and speed up your computer. If you're on a lower-end or older PC, or just like the idea of ​​running a lighter operating system, this guide can help you fix the problem with a single command.

The traditional way to remove bloatware involves finding those useless apps and uninstalling them one by one. To some extent it still works, as long as you can find the app listed under Settings > Apps > Apps & features. Here, find a bloatware app, click the three-dot Menu button, and select the “Uninstall” option. Confirm and follow the procedure.

But it doesn't work for hidden apps, system apps and background processes that Microsoft hacks into the operating system and that's where an automated script can save your life. The Sycnex developer's Windows10Debloater script works on both Windows 10 and Windows 11 and gives you a one-click option to remove all bloatware or a custom subset from the bloatware list.

Before proceeding, you should be aware of the inherent risks of running scripts from GitHub or another source, especially those that require high permissions. This particular script wants administrative privileges which, if written with malicious or shoddy content, could negatively affect your system.

At the time of writing, there are no reports of this script being approximately years old, being compromised, or having any unintended effects. That said, it's important to always exercise caution before diving. Read the scripts before running them on your PC to see if it is actually running the processes it advertises. If you see any strange or suspicious actions, such as changing privacy and security settings or adding new users, it's best to avoid running this script.

To get started, go to the start menu, search for "Windows PowerShell", right-click the result and select "Run as administrator".

In the PowerShell window, type the command below and press the Enter key. It will automatically download files from the GitHub repository and create a system restore point, in case something goes wrong.

iwr -useb https://git.io/debloat|iex



Entering this command will launch a new program where you will see options to disable Cortana, One Drive, etc.





Click the "Customize Blocklist" button and select the bloatware you want to remove. Save the file, then click the "Remove Bloatware with Custom Block List" button.

The script will run and uninstall the selected applications. Once done, reboot your PC and you'll be running a little lighter and hopefully a little more productive.
